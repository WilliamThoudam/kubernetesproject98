It will help to integrate wpForo with PeepSo.

== Description ==
1) Admin can set activity settings and activity content.
It will also allow users to enable their wpForo activity.
2) Admin can enable or disable forum activities inside PeepSo User Profile section.

If you need additional help you can contact us for [Custom Development](https://wbcomdesigns.com/contact/).


== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/peepso-wpforo' directory.
2. Activate the plugin through the \'Plugins\' screen in WordPress.
